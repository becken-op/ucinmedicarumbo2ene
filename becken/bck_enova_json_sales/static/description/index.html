Enova Json Import


Create quotations throw json files. Use this module to import json files coming from the Enova system. For each processed file, it will generate a quote with the data coming from your client. You will simply need to configure the ftp configuration properties as well as schedule the scheduled task to specify the frequency of execution. Your vendors will transparently see the quotes placed by your client.
