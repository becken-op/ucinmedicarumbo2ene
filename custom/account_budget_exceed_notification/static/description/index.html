Account Budget Exceed Practical Amount Notify

Account Budget Exceeds Alert Notification in Odoo

(Only for Odoo Enteprise Edition)

This app send email notification if budget practical amount exceed planned amount.

Main Features

  • Allow you to configure notification users on budget positions and on budget form.
  • If budget practical amount exceeds budget planned amount then system will send email notification to listed users.
  • For more details please check below screenshots and watch the video.

Edition Compatibility:


Enterprise Odoo.sh
