Customer Target

Overview

Provide the ability to set target based on customer from sales invoices within duration.

Features

  • Set customer target duration.
  • Appear amount in company currency.
  • analysis reports for all customers.

Configuration

Set customer target

How to Work

Create and validate invoice / credit note for customer or any contact related to customer

System will calculate actual from any invoice has commercial partner

Notes: System calculate untaxed amount and if invoice / credit note open or paid.

Get reporting for all customers in pivot and list view

Notes:This module is tested and works on Odoo Community and Enterprise Edition.
