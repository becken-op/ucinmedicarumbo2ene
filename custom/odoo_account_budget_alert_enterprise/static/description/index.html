Account Budget Alert and Warning On Sale, Purchase and Invoices

Only Compatible with Odoo Enterprise Edition

This app allows you to raise warnings and alerts for budget exceed situations on Sales Order, Customer Invoices and Purchase Orders.

For community edition of modules you can check and buy https://apps.odoo.com/apps/modules/14.0/odoo_account_budget_alert/ if you are working on an odoo community.

Highlights:

  • Odoo Account Budget Alert Message.
  • Odoo Account Budget Warning Message.
  • Budget Alerts and Warnings in Odoo on Sales Order.
  • Budget Alerts and Warnings in Odoo on Purchase Order.
  • Budget Alerts and Warnings in Odoo on Customer Invoice / Bills.

Main Features:

  • Account Invoice
    • alert type is Stop / Restrict, popup alert generated for scheduled budget and stop validate process of Account Invoice.
    • When alert type is Warn Message, only popup message generated for scheduled budget.
    • When alert type is Ignore, In this case no alert generated for scheduled budget.
  • Vendor Bill
    • When alert type is Stop / Restrict, popup alert generated for scheduled budget and stop validate process of Vendor Bill.
    • When alert type is Warn Message, only popup message generated for scheduled budget.
    • When alert type is Ignore, In this case no alert generated for scheduled budget.
  • Sale Order
    • When alert type is Stop / Restrict, popup alert generated for scheduled budget and stop confirm process of Sale Order.
    • When alert type is Warn Message, only popup message generated for scheduled budget.
    • When alert type is Ignore, In this case no alert generated for scheduled budget.
  • Purchase Order
    • When alert type is Stop / Restrict, popup alert generated for scheduled budget and stop confirm order process of Purchase.
    • When alert type is Warn Message, only popup message generated for scheduled budget.
    • When alert type is Ignore, In this case no alert generated for scheduled budget.
